# Configuración Inicial.

## 1. Setup del proyecto | Creación del entorno

```conda create -n reto_u1 python=3.11```

## 2. Activación del entorno

```conda activate reto_u1```

## 3. Instalación de Librerías a utilizar

```conda install numpy pandas sqlalchemy```

## 4. Verificación de instalación (ver versiones instaladas)

```conda list "^(numpy|pandas|sqlalchemy)$"```

## 5. Instalando dependencias

```pip install -r requirements.txt```
