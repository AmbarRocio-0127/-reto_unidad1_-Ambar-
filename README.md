# Reto Data Analist Junior - Análisis a TechVentas S.A.S

Análisis de ventas a la empresa TechVentas S.A.S respecto a los ultimos 6 meses. Con los Datos crudos se realizan los procesos correspondientes para proceder a entenderlos, consultarlos y limpiarlos para extraer el valor y dar respuestas de negocio.

## Tecnologías utilizadas

- Python
- SQLAlchemy
- Numpy
- SQL
- Pandas
- Jupyter Notebook
- Git
- GitHub


## Funcionalidades

- Consultas SQL.
- Joins.
- Subconsultas.
- Agregaciones (`GROUP BY`, `HAVING`).
- Uso de SQLAlchemy.
- Visualización de resultados con Pandas.
